============
Installation
============

Dépendances
===========

Avant de commencer, les dépendances suivantes sont nécessaires : 

.. code-block:: bash

        $ sudo apt install gawk wget git-core diffsat unzip textinfo gcc-multilib build-essential chrpath socat cpio python python3 python3-pip python3-pexpect xz-utils debianutils iputils-ping libsdl1.2-dev xterm

Chaine de cross compilation *STANDARD*:

.. code-block:: bash

        $ apt-get install gcc-arm-linux-gnueabi


Poky :
Nous n'utilisons pas la dernière version de Poky car nous aurons besoins de Layers qui ne sont supporté que dans une version précédente.

.. code-block:: bash

        $ git clone -b rocko https://git.yoctoproject.org/git/poky
