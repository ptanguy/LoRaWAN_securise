Introduction¶

Ce document est la documentation pour le Mini-Projet Home automation box. Nous devons réaliser une distribution Linux à partir d’un environnement de développement. Cet environnement permettra ensuite à notre client de développer un systéme de box domitique à partir de notre distribution. Notre distribution devra fonctionner sur 2 cartes différentes :

  • Hummingboard
  • Raspberry Pi 3 b+

La distribution doit être fournit avec un programme permettant de lire des valeurs d’un capteur I2C et d’écrire celles-ci sur le port série de la carte.
